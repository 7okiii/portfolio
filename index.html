Dynamic dashboard

HTML, Sass, Bootstrap5, Javascript, jQuery, PHP, MySQL, GitHub

A dynamic dashboard for the educational institution. Dashboard and content pages are created by HTML, Sass, Javascript and jQuery for the front-end. PHP to routine pages, Hash and salt technique for secured data storing.
Allow school administrators to register new students and teachers, able to publish new courses and appoint teachers for each course, and appoint new admin users. For class size, set maximum and minimum capacity, allows teachers to mark and leave comments for students’ learning and progress, students can see their marks and comments from teachers. And also enroll for a course and check the price and detail for each course.

Online shopping system

HTML, Sass, Angular JS, jQuery

An in-class project which uses Angular JS. HTML, Sass and jQuery for front-end. Angular JS logic to validate each input box, by using ng-model access to the element and display a modal box for each item when the item is clicked. jQuery to make a fade-out animation when an item is added to a shopping cart.
Customers can choose the number of items and add them to a shopping cart. And check the details and total price, edit the number of items and delete them from the shopping cart.

Responsive website

HTML, Sass, Bootstrap, Javascript, jQuery

A fully responsive landing page for a musician uses Bootstrap and the media query method. Javascript to validate each input box by Regular expression. Display different contents depending on what button is clicked.
The viewer can see information about the musician, watch a music video by a provided link and check the event and concert date. Contact musician through the validated contact form.
